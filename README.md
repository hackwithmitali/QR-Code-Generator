This is a web-based QR Code Generator built using Python, Flask, HTML, CSS, and QRCode libraries.

It allows users to:

2. Choose color and size
3. Add preset social media logos (Instagram, YouTube, LinkedIn)
4. Upload custom logos
5. Download QR as PNG/PDF
6. Share via WhatsApp or Email

Tech Stack:
1. Frontend:- Python
2. Flask
3. HTML
4. CSS
5. qrcode library
6. Pillow
7. ReportLab
